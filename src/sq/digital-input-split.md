# Digital Input Split

## With a Stagebox

::: warning
Doing a digital split with a stagebox requires that the monitor console has a second SLink port be installed using the SLink I/O Card
:::

**On the monitor Console:**

1. Press the `I/O` Button
2. Select `Inputs` on the left side
3. Select `Input Channels` sub menu on the left side
4. Select `SLink` on the top menu
5. Setup your patching for your input channels
6. Select `Tie Lines`
7. Select `SLink` on the left side
8. Select `I/O Port` on the top menu
9. Map up to the first 64 channels "1-to-1"

**On the FOH Console:**

1. Press the `I/O` Button
2. Select `Inputs` on the left side
3. Select `Input Channels` sub menu on the left side
4. Select `SLink` on the top menu
5. Setup your patching for your input channels

::: info
FOH will only have trim for gain control while Monitors will need to make sure to set gain correctly.
:::

## Without a Stagebox

**On the monitor Console:**

1. Press the `I/O` Button
2. Select `Inputs` on the left side
3. Select `Input Channels` sub menu on the left side
4. Select `Local` on the top menu
5. Setup your patching for your input channels
6. Select `Tie Lines`
7. Select `Local` on the left side
8. Select `I/O Port` on the top menu
9. Map up to the first 64 channels "1-to-1"

**On the FOH Console:**

1. Press the `I/O` Button
2. Select `Inputs` on the left side
3. Select `Input Channels` sub menu on the left side
4. Select `SLink` on the top menu
5. Setup your patching for your input channels
