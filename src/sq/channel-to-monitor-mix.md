# Sending a Channel to a Monitor Mix

## All Inputs to a Single Mix

This option is for occansions where you need to build an Aux mix 1 at a time. Similar to a "Sends on Faders" Button on an X32/M32.

1. Press the button on the right side for the monitor mix you want to send the channel to, Once this button is pressed your board is displaying the mix for that monitor mix.
2. Adjust the fader to the desired level for that monitor mix.
3. Press the same button on the right side OR press the `LR` button above the `MainLR` Mute Button to return the regular mix.

## A Single Input to all Mixes

This option shows all the send levels of the selected input. This options is useful during soundcheck when you are going through 1 channel at a time and need to send that channel to multiple mixes at the same time.

1. Press the `Select` Button on the channel you want to send to the aux mixes.
2. Press and hold the Blue `CH to All Mix` Button on the left side neat the LCD stribble strips.
3. While holding that button, adjust the level of the selected channel to each mix.
4. Release the button when done to return the channels.
